QFQ Extension
=============================================================

.. only:: html

	:Classification:
	      			qfq

	          	:Version:
	          		|release|

	:Language:
	          		en

	          	:Description:
	      	    		The extension offers support to:

	      	    		 * Create HTML Forms by clicking them together,
	      	    		 * Create reports based an SQL queries. The SQL can be nested and offers support for any kind of tags.
